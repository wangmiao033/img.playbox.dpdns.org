测试版安装：只看这 3 步

1. 下载并解压

点击上方“下载测试版插件”，下载后右键解压。解压后找到包含 manifest.json 的文件夹。

2. 加载到 Chrome

打开 chrome://extensions/，右上角开启“开发者模式”，点击“加载已解压的扩展程序”。

3. 选择插件文件夹

选择刚才解压出的、包含 manifest.json 的文件夹。打开任意网页后点右上角“素材猎手”图标使用。

当前还没上架 Chrome Web Store，所以测试版不能网页自动安装；上架后会变成“添加至 Chrome”一键安装。
