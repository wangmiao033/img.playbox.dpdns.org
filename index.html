插件安装方法

1. 下载插件包

点击页面上方“下载 Chrome 插件包”，下载的是精简插件分支，不再包含官网、部署文档和其他杂项。

2. 解压文件

解压后进入文件夹，里面应能看到 manifest.json、popup.html、background.js、assets 等插件文件。

3. 打开扩展管理

Chrome 地址栏输入 chrome://extensions/，右上角打开“开发者模式”。

4. 加载已解压扩展

点击“加载已解压的扩展程序”，选择刚才解压出来、包含 manifest.json 的文件夹。

5. 打开素材页面

进入游戏官网、素材页、文章页等普通网页，然后点击浏览器右上角“素材猎手”图标。

6. 筛选并下载

按格式、Min size、素材类型筛选图片，勾选后点击“下载选中”或“下载当前筛选”。

注意：当前还没上架 Chrome Web Store，所以不能网页自动安装。Chrome 只允许商店版一键安装；测试版必须先解压，再用开发者模式加载。
