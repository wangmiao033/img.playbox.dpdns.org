# 素材猎手 - Image Hunter

素材猎手是一款网页图片批量下载工具，适合运营、设计、游戏发行和内容编辑快速整理网页中的图片素材。

## 核心功能

- 一键扫描当前网页图片
- 自动识别 JPG、PNG、WEBP、SVG、GIF、AVIF 等格式
- 根据图片尺寸过滤小图标和无效素材
- 支持单张下载、多选下载和当前筛选结果下载
- 自动生成规范文件名，便于后续归档
- 识别 icon、banner、splash、portrait、landscape 等素材类型

## 适合场景

- 游戏渠道素材整理
- 网页图片批量归档
- 竞品素材参考收集
- 宣传图、icon、banner 快速下载
- 运营活动素材管理
